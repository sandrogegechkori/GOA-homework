თემის სახელი მოკლე აღწერა
HTML HTML არის პროგრამირების ენა რომელიც გამოიყენება ვებ-გვერდების სტრუქტურის შესაქმნელად.
CSS CSS გამოიყენება ვებ-გვერდების ვიზუალური დიზაინისა და სტილისტიკის შესაცვლელად.
JavaScript JavaScript არის პროგრამირების ენა, რომელიც გამოიყენება ვებ-გვერდებზე ინტერიაქტივობის და ფუნქციონალობის დასამატებლად.
Python Python არის მაღალი დონის პროგრამირების ენა, რომელიც ცნობილია თავისი გამარტივებული სინტაქსით და მრავალფეროვანი აპლიკაციებით.
