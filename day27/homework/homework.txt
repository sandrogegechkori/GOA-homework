კუთვნილება
ეს არის CSS თვისება, რომელიც აღწერს სტილს.
მაგალითად: color, font-size, margin.
კუთვნილება განსაზღვრავს, თუ რა ტიპის ცვლილება უნდა განხორციელდეს ელემენტზე.

მნიშვნელობა
ეს არის კუთვნილების კონკრეტული მნიშვნელობა, რომელიც განსაზღვრავს, როგორ უნდა გამოიყურებოდეს ელემენტი.
მაგალითად: red, 16px, bold.

CSS წესი შედგება სელექტორისა და დეკლარაციების ბლოკისგან.

 Selector (სელექტორი) სელექტორი ირჩევს HTML ელემენტს, რომელზეც მოქმედებს CSS წესი.

 CSS Inheritance (მემკვიდრეობა)
მემკვიდრეობა ნიშნავს, რომ ზოგი თვისება ავტომატურად გადაეცემა შიდა ელემენტებს.
მაგალითად: ტექსტთან დაკავშირებული თვისებები (color, font-family) შვილობილი ელემენტებისთვის მემკვიდრეობით გადადის.
